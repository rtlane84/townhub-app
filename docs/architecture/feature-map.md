## Shared boundaries

- Database schema and exports: `lib/db/src/schema` and `lib/db/src/index.ts`.
- API contract: `lib/api-spec/openapi.yaml`; generation config in `lib/api-spec/orval.config.ts`.
- Generated React client: `lib/api-client-react/src/generated`; auth/base fetch bridge in `lib/api-client-react/src/custom-fetch.ts`.
- Generated and hand-written validation: `lib/api-zod/src/generated` and sibling shared rule modules.
- API unit tests live next to `artifacts/api-server/src/**/*.test.ts`; frontend rule tests live in `artifacts/townhub/src/lib/*.test.ts`; browser workflows live in `tests/e2e`.
